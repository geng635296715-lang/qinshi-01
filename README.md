# Qing Shi Ji Frontend

Static frontend bundle for the Qing Shi Ji homepage.

Entry file:
- `index.html`

Key local files:
- `styles.css`
- `ai-chat-orb.js`
- `assets/`

Notes:
- This is a plain static site bundle with relative asset paths.
- The AI chat background effect loads `ogl` from a CDN inside `ai-chat-orb.js`.
